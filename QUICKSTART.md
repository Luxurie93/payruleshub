## 🚀 Deploy in 5 Minutes

### Option 1: Vercel (Easiest)
```bash
npm install -g vercel
vercel
# Follow prompts → Done!
```

### Option 2: Netlify (GitHub)
1. Push to GitHub
2. Connect repo to Netlify
3. Auto-deploys on push

### Option 3: Any Static Host
```bash
npm run build
# Deploy `.next` directory anywhere
```

## 📋 Next Steps

### 1. Customize (5 min)
- Update domain in `app/sitemap.ts` & `app/robots.ts`
- Update site name in `app/layout.tsx`
- Change brand colors in `tailwind.config.ts` (optional)

### 2. Deploy (5 min)
- Push to production (Vercel, Netlify, etc.)
- Add custom domain
- Enable HTTPS (automatic)

### 3. Index with Google (5 min)
- Go to Google Search Console
- Add property → Verify ownership
- Submit sitemap from `/sitemap.xml`
- Request indexing for all pages

### 4. Set Up Analytics (5 min)
- Create Google Analytics 4 property
- Update GA4 ID in `app/layout.tsx`
- Set up AdSense (after 30 days, 15+ pages ✅)

### 5. Monitor & Iterate (ongoing)
- Check Search Console weekly
- Optimize low-CTR page titles
- Add state-specific overtime pages (high-traffic lever)
- Monitor calculator usage in GA
